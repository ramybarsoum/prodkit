# Wiki

Prodkit maintains this folder. You review it, the LLM writes it.

Pages are created automatically when you run skills or ingest sources. Each page is cross-linked and categorized in INDEX.md.

Page types:

| Type | File pattern | What it captures |
|------|-------------|-----------------|
| Entity | `entity-<name>.md` | Person, company, product, or team |
| Concept | `concept-<name>.md` | Framework, pattern, or domain concept |
| Feature | `feature-<name>.md` | Aggregated spec, sizing, research, decisions for one feature |
| Comparison | `comparison-<topic>.md` | Side-by-side analysis of competitors or approaches |
| Synthesis | `synthesis-<topic>.md` | Cross-cutting insight connecting multiple sources |
| Decision | `decision-<topic>.md` | Why a decision was made, alternatives, trade-offs |

INDEX.md is the front page. Prodkit reads it first to navigate the wiki.

Browse with Obsidian for graph visualization of links.
