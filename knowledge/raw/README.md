# Raw Sources

Drop your original documents here. Prodkit reads them but never modifies them.

Examples: interview transcripts, strategy decks, analytics exports, competitor screenshots, research papers, Slack threads, email chains.

Subfolders are fine for organization:

```
raw/
  interviews/
  competitor-decks/
  analytics-exports/
  strategy/
```

This folder is the source of truth. Wiki pages link back here for verification.
