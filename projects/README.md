# Projects

One folder per feature or initiative. Prodkit creates these when you run skills like `/feature-spec-interview` or `/launch-checklist` for a named feature.

```
projects/
  checkout-redesign/
    feature-spec.md
    review-panel-report.md
    launch-checklist.md
  onboarding-v2/
    feature-spec.md
```

General knowledge (research synthesis, competitor analysis, decisions, impact sizing, North Star) goes in `knowledge/wiki/` instead.
